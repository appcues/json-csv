# json-csv

A command-line utility and Ruby gem for converting JSON files to CSV,
and CSV files to JSON (soon).


## Why json-csv?

* Customizable: field whitelist and blacklist, max nested data depth,
  line endings, etc

* Zero dependencies outside core Ruby 2.x

* Easy install
